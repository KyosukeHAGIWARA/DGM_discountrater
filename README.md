# DGM_discountrater

## Description
デジマートのセール商品に割引率情報を付加するChrome Extensionです。  
なんのひねりもない安直な名前ですね。

## Usage

ストアにあげるつもりはないので使いたければここをクローンするとよいとおもいます。

1. clone or ZIP Download
1. 展開する
1. 開発中のchrome extensionを追加する流れでディレクトリをchromeに追加
1. デジマート商品に割引率が追加
